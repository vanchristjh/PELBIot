## 🔧 QUICK START

### 1. Install Dependencies
```bash
npm install
npm install ioredis swagger-ui-express swagger-jsdoc
```

### 2. Setup Environment
```bash
# .env file
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_DB=0
```

### 3. Start Services
```bash
# Start Redis
redis-server

# Start backend
npm run dev

# Frontend should be on port 3000
# Backend API on port 5000
# API Docs on http://localhost:5000/api-docs
```
