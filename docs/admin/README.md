## 👤 User Roles

### Super Admin
- Full system access
- Create/edit/delete users
- Modify all settings
- View audit logs
- System-wide configuration

### Admin
- Manage general users
- Create reports
- Modify alerts
- Configure features
- Limited system settings

### Moderator
- View data and reports
- Acknowledge alerts
- Limited configuration
- No user management
- Monitoring only
